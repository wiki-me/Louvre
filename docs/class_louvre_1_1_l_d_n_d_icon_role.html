Detailed Description

Drag & drop icon role for surfaces.

The LDNDIconRole class is a role for surfaces that allows the compositor to use them as icons for drag & drop sessions.
Clients create the role through the start_drag request from the Wayland wl_data_device protocol interface.
The LDNDIconRole role used in a drag & drop session can be accessed from LDNDManager::icon().

See also

virtual const LPoint& rolePos ( ) const

Position of the surface given the role.

The position of the icon given the role is calculated by subtracting the hotspot from the surface position.
This method can be reimplemented to change the positioning logic of the surface given the role.

Default implementation

const LPoint &LDNDIconRole::rolePos() const
{
m_rolePos = surface()->pos() - hotspot();
return m_rolePos;
}
